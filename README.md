## Installation
Run These commands in the terminal: 
npm init

npm install mysql2

npm install sequelize

npm install dotenv

Run the following at the root of the project :

mysql -u root -p

Enter PW when promted

source db/schema.sql

quit

npm run seed

lastly run npm start
